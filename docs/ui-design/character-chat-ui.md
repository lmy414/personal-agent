# Character Chat UI 设计文档

## 1. 概述

AI 角色扮演聊天界面，以动漫角色立绘为全屏背景，左侧为角色信息与设置面板，右侧为聊天对话面板。

## 2. 布局架构

### 2.1 整体布局 (Grid)

```
+--------------------------------------------------+
| Toolbar (60px)                                   |
+----------+---------------------------------------+
| Sidebar  | Center + Chat Panel                   |
| (320px)  |                                       |
|          |  Center: 角色立绘背景                 |
|          |  Chat:  浮动于右侧                    |
+----------+---------------------------------------+
```

### 2.2 区域划分

| 区域 | 宽度 | 定位 | 说明 |
|------|------|------|------|
| Toolbar | 100% | fixed top | 工具按钮行 |
| Sidebar | 320px | fixed left | 角色信息 + 设置 |
| Center | flex-1 | relative | 角色立绘背景 |
| Chat Panel | 480px | absolute right | 对话消息列表 |

## 3. 组件清单

### 3.1 Toolbar
- 汉堡菜单按钮
- 角色名称显示
- 工具图标组：设置、主题、字体、剪贴板、表情、用户、全屏

### 3.2 Sidebar
- **角色卡片**：头像(圆形)、名称、令牌信息
- **操作按钮**：收藏、编辑、复制、删除
- **描述区域**：可滚动的文本框，显示角色设定
- **设置面板**：
  - Context Size 滑块 (0-8192)
  - Streaming 开关
  - Auto-purge API context 开关
  - Auto-jailbreak 开关
  - Send character note 开关
- **Jailbreak 消息输入框**

### 3.3 Center (角色展示区)
- 全屏背景图（角色立绘）
- 背景图覆盖整个视口
- 暗色遮罩确保文字可读性

### 3.4 Chat Panel
- **消息列表**：垂直滚动
  - 角色消息：左侧头像 + 名称 + 内容
  - 用户消息：左侧头像 + "You" + 内容
- **输入框**：底部固定
  - 菜单按钮
  - 文本输入
  - 附件按钮

## 4. 样式规范

### 4.1 颜色

| Token | 值 | 用途 |
|-------|-----|------|
| --bg-primary | #1a1a2e | 主背景 |
| --bg-secondary | #16213e | 面板背景 |
| --bg-tertiary | #0f3460 | 输入框背景 |
| --text-primary | #e94560 | 强调色（粉红）|
| --text-secondary | #eaeaea | 主文字 |
| --text-muted | #a0a0a0 | 次要文字 |
| --border | rgba(255,255,255,0.1) | 边框 |
| --accent | #e94560 | 按钮/开关激活 |

### 4.2 字体
- 主字体：system-ui, -apple-system, sans-serif
- 聊天内容：14px / line-height 1.6
- 标题：16px / font-weight 600

### 4.3 间距
- 面板内边距：16px
- 组件间距：12px
- 消息间距：16px
- 圆角：8px（卡片）、12px（消息气泡）

## 5. 交互设计

### 5.1 开关组件
- 自定义 checkbox，激活时显示粉色背景 + 白色对勾

### 5.2 滑块组件
- 轨道：深灰色背景
- 滑块：粉色圆形 thumb
- 数值显示在右侧

### 5.3 消息气泡
- 角色消息：深色半透明背景 + 左侧头像
- 用户消息：略浅背景 + 左侧头像
- 悬停时轻微高亮

### 5.4 输入框
- 底部固定
- 聚焦时边框高亮
- 支持多行文本

## 6. 响应式

- 桌面 (>1024px)：完整三栏布局
- 平板 (768-1024px)：隐藏 Sidebar，Toolbar 简化
- 手机 (<768px)：仅保留 Chat Panel 全屏
