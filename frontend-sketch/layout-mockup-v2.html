NPR 渲染管线备忘

当前使用的 ILM-based NPR 流程：

  1. TGA 贴图导入需关闭 Channel Pack
  2. A 通道 = 描边遮罩，R 通道 = 阴影渐变
  3. Half Lambert 比 Ramp Map 更适合本项目的风格化需求

参考代码：

vec4 ilm = texture(ilm_tex, uv);
float outline = ilm.a;
float shadow = ilm.r;
float spec = ilm.g;

vec3 diffuse = half_lambert(normal, light_dir);
vec3 color = mix(shadow_color, base_color,
  smoothstep(0.3, 0.7, shadow * diffuse));

Godot 4.x, Blender 4.2, GLSL

## NPR 渲染管线备忘 当前使用的 **ILM-based NPR** 流程： 1. TGA 贴图导入需**关闭 Channel Pack** 2. A 通道 = 描边遮罩，R 通道 = 阴影渐变 3.
