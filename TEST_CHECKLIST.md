## 澪号 Harness

- [ ] 扩展加载：F12 Console 搜索 `[pa-mio]`，应看到 `澪号 Harness 已加载`
- [ ] 基础对话：发送"你好"，澪号应回复短句、不叠甲风格的消息
- [ ] 外观确认：问"你长什么样"，应回复 163cm/灰发/红瞳/双马尾
- [ ] 傲娇模式：帮忙后说"谢谢"，应回复类似"不是你让我看的吗"
- [ ] 工具调用：发送 `list_directory .`，目录列表应正确换行（无竖向排布 bug）
- [ ] 工具后风格保持：工具调用后继续对话，澪号不应变成 coding agent 语气
- [ ] 历史渲染：切换会话再切回来，工具调用块正常显示
- [ ] 叠甲检测：发起需要技术判断的对话，确认澪号不叠甲
- [ ] 记忆写入：完成 >10 轮对话后，检查 `~/.personal-agent/mio_memories.json` 是否有新条目
